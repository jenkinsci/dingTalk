# 快速开始

::: warning

请确保你的 Jenkins 版本 >= 2.176.4

:::

## 安装插件

在 `Manage Plugins` 安装 [DingTalk](https://plugins.jenkins.io/dingding-notifications/)。

## 机器人配置

在 `Configure System` 中找到 `钉钉配置` 选项卡，根据自己的需求选择 `通知时机`，然后添加机器人即可。
